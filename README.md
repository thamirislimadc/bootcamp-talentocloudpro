# Bootcamp Talento Cloud Pro

## Descrição
- Repositório com arquivos criados nas aulas do Bootcamp Talento Cloud Pro, criado pela Proz Educação em parceria com a Amazon Web Services (AWS).
- A formação de divide em duas trilhas: "Introdução à programação com foco em Front-End" e "Cloud Practitioner".
- Conteúdo programático: lógica de programação, introdução a programação, HTML, CSS, JavaScript, Git, GitHub, Banco de Dados e Cloud.
- Duração: 4 meses
- Modelo: aulas ao vivo e assíncronas
- Carga horária: 300h
